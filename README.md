# resurrection-man
grave robbing and other fun things

# dependencies
[pyenv](https://github.com/pyenv/pyenv#automatic-installer)

[pipenv](https://pipenv.pypa.io/en/latest/)

# getting started
`pipenv install`

`make run`

# editing assets
`make edit`

# type checking
`make types`

# uml
`make uml`

might require graphviz

`brew install graphviz`

or

`apt-get install graphviz`
